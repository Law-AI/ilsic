# ilsic
This repository will hold the codes and data files for paper "ILSIC: Corpora for Identifying Indian Legal Statutes from Queries by Laymen" (EACL 2026)

## Dataset Download

Download the `dataset` folder from Google Drive using the link below:
 https://drive.google.com/drive/folders/1m_tU6Cb55Q5mVQEImFUp-cUSVZzoHzfd?usp=sharing

After downloading, place the `dataset` folder in the root directory of the project.
